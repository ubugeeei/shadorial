# Shadorial

Interactive shader tutorial for learning GLSL and WebGL2 hands-on.

https://ubugeeei.github.io/shadorial

## Chapters

19 chapters covering shader fundamentals, wave functions, shapes, generative art, and Three.js integration.

## Tech Stack

Vue 3 / TypeScript / Vite / WebGL2 / Three.js / Monaco Editor

## Development

```bash
pnpm install
pnpm dev
```
